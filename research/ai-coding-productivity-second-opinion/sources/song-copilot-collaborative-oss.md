# Song et al. Collaborative Open-Source Copilot Study

## Source Type

arXiv preprint using GitHub proprietary Copilot usage data combined with public OSS data. Independence is limited by proprietary platform data access.

## Method

Observational/quasi-experimental study of GitHub Copilot in collaborative open-source development.

## Key Evidence

- Copilot use is associated with a 5.9% increase in project-level code contributions.
- The gain is decomposed into a 3.4% rise in coding participation and a 2.1% increase in individual productivity.
- Coordination time increases by 8% due to more code discussions.
- Peripheral developers see smaller contribution gains and larger coordination-time increases than core developers.

## Caveats

This is not task-level completion-time evidence. Project-level contribution gains do not necessarily mean experienced developers finish their own work faster.

## Practical Implication

AI can increase contribution volume while increasing coordination burden. Role and familiarity matter.
