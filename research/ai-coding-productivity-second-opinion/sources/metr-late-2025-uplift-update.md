# METR Late-2025 Uplift Update

## Source Type

Independent nonprofit research update. It is a methodological update rather than a clean, final RCT result.

## Key Evidence

- METR says its earlier early-2025 slowdown result no longer fully reflects current AI impact.
- A later experiment with 57 developers across 143 repositories and 800+ tasks produced raw estimates suggesting possible speedup, but METR states the signal is unreliable.
- Reported estimates: the original-developer subset showed an estimated 18% speedup with a confidence interval from 38% speedup to 9% slowdown; newly recruited developers showed an estimated 4% speedup with a confidence interval from 15% speedup to 9% slowdown.
- METR identifies selection effects: some developers did not want to participate if they might have to work without AI, and 30% to 50% reported not submitting tasks they did not want to do without AI.
- Agentic tools made timing harder because developers may work on unrelated tasks while agents run.

## Caveats

This is not strong evidence for a precise current speedup. METR itself calls the new data unreliable for the current productivity effect.

## Practical Implication

The most current METR update weakens any claim that the early-2025 19% slowdown is a timeless estimate. It does not replace it with a well-measured general speedup.
