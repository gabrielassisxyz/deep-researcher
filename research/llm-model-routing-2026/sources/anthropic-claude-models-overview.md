# Anthropic Claude Models Overview

## Summary

Anthropic's model overview verifies the current Claude frontier lineup and model identifiers used in this dossier: `claude-fable-5`, `claude-opus-4-8`, `claude-sonnet-5`, and related aliases. The page says users who are unsure should start with Claude Opus 4.8 for complex agentic coding and enterprise work, and use Claude Fable 5 for workloads needing the highest available capability. It also lists multimodal input support for current Claude models and notes availability through the Claude API and major cloud providers.

## Claims Used

- Claude Fable 5 is Anthropic's most capable widely released model.
- Claude Opus 4.8 is recommended by Anthropic for complex agentic coding and enterprise work.
- Claude Sonnet 5 is part of the latest Claude model lineup.
- Current Claude models support text and image input with text output.
- Claude Opus 4.8 and Claude Sonnet 5 default to high effort.
- Claude Sonnet 4.6 is supported in later-model infrastructure and batch output contexts, but it is no longer the latest Sonnet model.

## Limitations

This is a vendor source. Capability claims are strong for availability, identifiers, and intended use, but benchmark-like quality claims require independent corroboration.
