# OpenAI GPT-5.5 Guide

## Summary

OpenAI's latest model guide identifies GPT-5.5 as the latest broadly documented OpenAI API model and describes it as the flagship model for complex reasoning and coding. The guide also says GPT-5.6 is available to select trusted partners in preview, not broad availability, and recommends smaller variants such as GPT-5.4 mini or nano when optimizing for latency or cost.

## Claims Used

- `gpt-5.5` is OpenAI's flagship model for complex reasoning and coding.
- GPT-5.4 mini and GPT-5.4 nano are the recommended lower-cost/lower-latency alternatives in OpenAI's own model family.
- GPT-5.6 is preview-only for trusted partners as of the access date.

## Limitations

The page is an official positioning document, not a model card with detailed benchmark methodology.
