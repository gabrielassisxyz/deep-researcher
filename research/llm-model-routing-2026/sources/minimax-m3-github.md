# MiniMax M3 Repository

## Summary

The MiniMax M3 repository describes MiniMax-M3 as a native multimodal model with 1M context, about 428B total parameters, and about 23B activated parameters. It says MiniMax Sparse Attention improves long-context inference speed relative to M2 at 1M context.

## Claims Used

- M3 is open-weight or intended for open-weight/local deployment in the MiniMax ecosystem.
- M3 has 428B total parameters and 23B active parameters.
- M3's architecture is designed for long-context and multimodal workloads.

## Limitations

Repository claims are primary for architecture and availability, but still vendor-authored for performance.
