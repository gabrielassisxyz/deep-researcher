# Google Gemini API Models

## Summary

Google's Gemini API model page verifies `gemini-3.1-pro-preview` and `gemini-3.5-flash`. It describes Gemini 3.1 Pro Preview as advanced intelligence for complex problem solving, agentic work, and coding. It also marks preview models as usable but more volatile than stable models, with more restrictive rate limits and a shorter deprecation notice than stable models.

## Claims Used

- `gemini-3.1-pro-preview` is the exact public model family entry for Gemini 3.1 Pro in this research.
- `gemini-3.5-flash` is a stable model entry.
- Gemini 3.1 Pro is positioned for complex problem solving, agentic capabilities, and coding.
- Preview status reduces production confidence compared with stable models.

## Limitations

The model list verifies availability and official positioning; it does not by itself prove cost-effective superiority for a task.
