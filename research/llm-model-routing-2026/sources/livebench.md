# LiveBench

## Summary

LiveBench is an ICLR 2025 spotlight benchmark designed to reduce contamination by regularly releasing new questions and using verifiable objective ground-truth answers instead of LLM judges. The site reports that the most recent version is LiveBench-2026-01-08, with categories including reasoning, coding, agentic coding, mathematics, data analysis, language, and instruction following.

## Claims Used

- LiveBench is a credible independent benchmark methodology source for coding, agentic coding, data analysis, reasoning, math, language, and instruction-following.
- It is particularly relevant to routing because it has task categories instead of a single global score.
- Objective ground-truth scoring reduces judge-model dependence compared with LLM-as-judge benchmarks.

## Limitations

The scraped leaderboard table did not include populated model rows in this pass. The dossier therefore uses LiveBench mainly for methodology and category framing, not for exact scores of the requested 2026 models.
