# Z.ai GLM-5 Repository

## Summary

The Z.ai GLM-5 repository verifies GLM-5.2 and GLM-5.2-FP8 checkpoints. It describes GLM-5 as built for complex systems engineering and long-horizon agentic tasks, with model sizes including 744B total parameters and 40B active parameters for GLM-5.2. It lists local deployment support through SGLang, vLLM, Transformers, KTransformers, Unsloth, and Ascend.

## Claims Used

- GLM-5.2 exists as an official Z.ai model.
- GLM-5.2 is open-weight/self-hostable through official checkpoint links.
- GLM-5.2 is positioned for complex systems engineering and agentic work.
- GLM-5.2 supports a reasoning-effort parameter and thinking controls.

## Limitations

Most benchmark performance claims in the repository are vendor-authored. They should not be treated as independent evidence without corroboration.
