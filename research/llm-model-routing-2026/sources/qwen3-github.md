# Qwen3 Repository

## Summary

The Qwen3 repository verifies Qwen3 model releases and Qwen3-2507 variants. It describes Qwen3-Instruct-2507 and Qwen3-Thinking-2507 improvements in instruction following, reasoning, coding, tool usage, multilingual knowledge, and 256K long-context support extendable to 1M.

## Claims Used

- Official Qwen3 releases exist with strong coding, reasoning, tool-use, and long-context positioning.
- Qwen3-2507 supports 256K context with extension to 1M.
- This source did not verify an exact `Qwen3.5` model name.

## Limitations

The requested `Qwen3.5` was not verified by this official source. OpenRouter verifies Qwen3.5 API entries, but official Alibaba/Qwen corroboration was not found in this pass.
