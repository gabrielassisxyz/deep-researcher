# Artificial Analysis Intelligence Benchmarking Methodology

## Summary

Artificial Analysis describes its Intelligence Index v4.1 as a text-only English suite combining nine evaluations across agents, coding, scientific reasoning, and general tasks. The index weights agents at 34%, coding at 24%, scientific reasoning at 24%, and general tasks at 18%. Included evaluations include GDPval-AA v2, tau3-Banking, Terminal-Bench v2.1, SciCode, AA-LCR, AA-Omniscience, Humanity's Last Exam, GPQA Diamond, and CritPt.

## Claims Used

- Artificial Analysis is useful for task-family evidence, not just generic ranking, because it separates agentic, coding, long-context, factuality, and scientific reasoning components.
- The methodology uses pass@1 in many evaluations and attempts consistent prompting, temperature, and retry rules.
- The Intelligence Index is text-only and English-only; multimodal and multilingual results are reported separately.
- GDPval-AA v2 uses frontier model judges including GPT-5.5, Gemini 3.1 Pro Preview, and Claude Opus 4.8, which is useful but introduces judge-model dependence for some agentic scoring.

## Limitations

The methodology is strong, but the index should not be used as a universal routing answer. The dossier uses it as methodological support for task taxonomy and benchmark caveats.
