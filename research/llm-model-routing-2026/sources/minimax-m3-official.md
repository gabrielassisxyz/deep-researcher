# MiniMax M3

## Summary

MiniMax's M3 page verifies MiniMax M3 and positions it as a coding and agentic model with 1M-context MSA and native multimodality. The page says the API supports up to 1M tokens with a guaranteed minimum of 512K, and includes vendor benchmark claims for BrowseComp, long-horizon paper reproduction, CUDA kernel optimization, and PostTrainBench.

## Claims Used

- MiniMax M3 is an official model.
- M3 supports up to 1M context, with a 512K guaranteed minimum according to MiniMax.
- M3 is natively multimodal and positioned for coding, agentic workflows, and long-range analysis.
- Vendor claims include BrowseComp 83.5, PostTrainBench 37.1, and a 9.4x autonomous CUDA optimization example.
- Vendor pricing claim includes $0.30 / MTok input and $0.06 / MTok blended cache cost.

## Limitations

The strongest performance examples are vendor demonstrations and should be treated as medium-confidence routing evidence until independently reproduced.
