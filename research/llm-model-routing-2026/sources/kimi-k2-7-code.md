# Kimi K2.7 Code

## Summary

Kimi's API documentation verifies `kimi-k2.7-code` and `kimi-k2.7-code-highspeed`. It calls Kimi K2.7 Code Kimi's most capable coding model to date, says it follows instructions more reliably in long contexts, improves long-horizon coding performance over K2.6, and offers a high-speed variant with about 180 tokens per second and up to 260 tokens per second in short-context scenarios.

## Claims Used

- `kimi-k2.7-code` is the exact verified model identifier corresponding to the requested Kimi K2.7-Code.
- The official context window is 256K according to Kimi's model list.
- The high-speed variant is relevant for high-volume implementation loops.
- Vendor claims indicate stronger instruction compliance and long-horizon coding than K2.6.

## Limitations

The performance deltas are vendor claims. They support routing hypotheses at medium confidence unless corroborated by independent benchmarks.
